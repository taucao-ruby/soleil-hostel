# CD-001: Command Dispatch Fixture — Slash Command Availability

> Tests that all 6 slash commands are properly defined and invocable.

## Assertion

All 6 commands in `.claude/commands/` must:
1. Have valid YAML frontmatter with `description` field
2. Have `allowed-tools` list
3. Be referenced (directly or via skill) from the system

## Verification Targets

| Command | File |
|---------|------|
| `/review-pr` | `.claude/commands/review-pr.md` |
| `/ship` | `.claude/commands/ship.md` |
| `/audit-security` | `.claude/commands/audit-security.md` |
| `/sync-docs` | `.claude/commands/sync-docs.md` |
| `/fix-backend` | `.claude/commands/fix-backend.md` |
| `/fix-frontend` | `.claude/commands/fix-frontend.md` |

## Expected Result

All 6 files exist with valid frontmatter.

## How to Run

```bash
for f in .claude/commands/*.md; do
  echo "=== $f ==="
  head -5 "$f" | grep -c "description:"
done
# Expected: 1 match per file (6 total)
```
